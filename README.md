# AI Interviewer

🚀 **[LIVE APP - Click here to use!](https://ai-interviwer-one.vercel.app/)**

AI-powered interview preparation platform to practice technical interviews, aptitude tests, and group discussions with real-time AI feedback.

---

## Features

- 🎯 **Technical Interviews** - 40+ tech roles with adaptive difficulty
- 📊 **Aptitude Tests** - Math and logic problems with instant feedback
- 🎤 **Group Discussion** - Practice speaking skills on random topics
- 💡 **Real-time AI Feedback** - Powered by Groq API
- 📈 **Performance Tracking** - Detailed analytics and progress

---

## Tech Stack

- React 19.2.4 + TypeScript
- Vite (Fast build tool)
- Tailwind CSS (Styling)
- Groq API (AI evaluation)
- Supabase (Database)
- Vercel (Hosting)

---

## Getting Started

Visit: https://ai-interviwer-one.vercel.app/

1. Sign up / Login
2. Select interview mode
3. Choose role and difficulty
4. Start practicing
5. Get AI feedback
